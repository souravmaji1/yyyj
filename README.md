# Intelliverse-X-User-Webfrontend

This repository contains the user facing web frontend for **IntelliVerse**, a gaming platform built with Next.js.

## Codebase Overview

- **src/app** - Next.js App Router structure
  - `(public)` - pages accessible to everyone
  - `(auth)` - authentication flows
  - `(protected)` - pages that require login
  - `(main)` - base layout wrapper
  - `apis` - server side route handlers
  - `utils` - small helpers used by the app
- **src/components** - reusable React components
  - `ui` - primitive UI elements (Button, Input, Dialog, etc.)
  - `layout` - global layout pieces such as navigation and footer
  - other folders provide domain specific widgets like product listings, cart or shop
- **src/store** - Redux Toolkit store configuration and slices. State is persisted using `redux-persist`.
- **src/constants** - project wide constants like colors and sizes
- **src/core** - miscellaneous utilities, configuration and helper functions
- **src/hooks** - shared custom hooks

## Development

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Run the linter:

```bash
npm run lint
```



## QR Code Login

- On desktop, open the QR login page to display a code.
- If you are already logged in on a mobile device, tap **Scan Desktop QR** on the login page to open the scanner at `/auth/qr-scan`.
- Scanning the desktop code will verify the session without re-entering credentials.
